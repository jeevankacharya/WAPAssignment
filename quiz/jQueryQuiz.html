1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

Ans : JQuery function is a function called from the root jQuery object that is used to convert DOM elements to
JQuery objects or grt JQuery object using selectors.

A JQuery object is array like which means that it contains zero or more indexes.

 

2. Write jQuery code to find all h1 elements that are children of a div element and make their background
color red.
Sample HTML:
<body>
<h1>abc</h1>
<br>
<div>
<h1>div-1</h1>
<h1>div-2</h1>
</div>
<h1>xyz</h1>
</body>

Ans :
<script>
$(document).ready(function(){
$('div h1').css("background-color","red");
});
</script>
